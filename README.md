# Project Atlas
Project Atlas is an agent-based economic simulation developed in Python. It models interactions between consumers and firms within a competitive smartphone market, using adaptive strategies, heterogeneous preferences and macroeconomic fluctuations.
The aim of Atlas is to create a simplified but dynamic model of a market economy. Consumers are represented as individual agents with different incomes, preferences and purchasing behaviours, while firms compete through decisions involving pricing, quality and brand strength.

Rather than modelling a single fixed market outcome, Atlas simulates how markets develop over time. Consumer choices influence firm performance, firm performance affects future strategies, and wider economic conditions can change the behaviour of both consumers and businesses.


Current Features

The current version of Atlas includes:

Agent-based consumer modelling
Heterogeneous consumer preferences
Income-based consumer segmentation
Utility-based purchasing decisions
Three competing smartphone firms
Economic boom, normal and recession periods
Variable market sizes based on economic conditions
Analysis of profit, revenue, costs and market share
Data visualisation using Python and Matplotlib
